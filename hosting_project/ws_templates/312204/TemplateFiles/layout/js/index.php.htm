Referrer Addon A simple addon which lists up the sites your visitors came from ...

Install:

- Upload all files to your FTP
- run referer-install.php
- include "_referer.php" into your index.php above the head and html tag
Code Snippet:
[...]
include("_mysql.php");
include("_settings.php");
include("_functions.php");
include("_referer.php");
[...]

- add a link in your admincenter.php like this:
<a href="admincenter.php?site=referer">Referer</a>
